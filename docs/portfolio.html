Resume Website

Tech Stack: Python, HTML, CSS, JS, Git, Bash

While creating this website, I had to understand how HTML, CSS, and JavaScript work together to create the website you see before you. Throughout the process, official bootstrap documentation, amongst other sites, were referenced for removing any blockers. Additionally, to facilitate the updating of the website, I wrote a python script to recompile the website with the new changes instead of doing it in each webpage one by one; this made the code much more efficient. Furthermore, version control, branching and merging concepts were used regularly with Git via the command line and uploaded to GitHub.

  • Date: September 2020
  • Client: Kickstart Coding Bootcamp
  • Category: Web Development

Account Manager

Tech Stack: Java, Git

This program simulates a returning user logging into their account. Alternatively, it allows a new user to register for an account. The code checks the password against a custom regular expression to increase the password strength. The code also implements an interface that has a set of methods that are required to be overwritten for the program to function. Furthermore, for demonstration purposes, at the end of the program it allows the user to see all the stored usernames and passwords. Source code can be examined on my GitHub.

  • Date: September 2020
  • Client: Dawit Gebremichael
  • Category: Backend Programming

Library Search & File IO

Tech Stack: Java, Git, SQL Concepts

This program reads in 5 CSV files that each contain information concering publishers, books, and authors. The remaining 2 CSV files, provide information that connect the primary keys in the first 3 files. Once all the information is loaded into memory, the program allows the user to search through the array of objects for a publisher, book, or author. If the provide name is found, the program outputs all the information. Furthermore, the code accounts for items with one-to-many relationships.

  • Date: October 2020
  • Client: Dawit Gebremichael
  • Category: Backend Programming
